## StaticSite

选择一个静态站点生成工具，我用的 [hexo](https://hexo.io/zh-cn/docs/)
跟着文档做就好了，主要是配置文件的修改，遇到问题才会去看源码。开始遇到的坑是图片的绝对路径错误
`//img/xxx.jpg` 生成后是这种路径，不过我添加了自定义域名后自动解决了这个问题

常用的hexo命令：

```none
hexo new post [name of post]
hexo generate
hexo serve
hexo clean && hexo deploy

```
